21. 배열 메서드 : join()

  • join() 메서드는 배열 요소를 문자열로 결합합니다.
  • 배열의 요소를 추가하여, 하나의 문자열로 반환합니다.
    • const arrNum1 = [100,100,300,400,500];
const text1 = arrNum1.join(''); //100200300400500
const text2 = arrNum1.join(' '); //100 200 300 400 500
const text3 = arrNum1.join('*'); //100*200*300*400*500
const text4 = arrNum1.join('-'); //100-200-300-400-500
const text5 = arrNum1.join('🦊'); //100🦊200🦊300🦊400🦊500

console.log(text1~5)

    22. 배열 메서드 : pop()

  • pop() 메서드는 배열 마지막 요소를 삭제합니다.
    • const arrNum2 = [100,200,300,400,500];
const arrPush2 = arrNum2.pop(); 

console.log(arrNum2) //[100,200,300,400]

    23. 배열 메서드 : push()

  • push() 메서드는 배열 마지막 요소에 요소를 추가합니다.
    • const arrNum1 = [100,200,300,400,500];
const arrPush1 = arrNum1.push(600);

console.log(arrNum1) // [100,200,300,400,500,600]

    28. 배열 메서드 : unshift()

  • unshift() 메서드는 배열 처음 요소에 요소를 추가
    • const arrNum1 = [100,200,300,400,500];
const arrPush1 = arrNum1.unshift(600);

console.log(arrNum1) // [600,100,200,300,400,500]
