배열 메서드 : push()

push() 메서드는 배열 마지막 요소에 요소를 추가하고, pop() 메서드는 배열 마지막 요소를 삭제합니다.

 const arrNum1 = [100,200,300,400,500];
const arrPush1 = arrNum1.push(600);

console.log(arrNum1) // [100,200,300,400,500,600]

const arrNum2 = [100,200,300,400,500];
const arrPush2 = arrNum2.push(); 

console.log(arrNum2) //[100,200,300,400]

배열 메서드 : shift()

unshift() 메서드는 배열 처음 요소에 요소를 추가하고, shift() 메서드는 배열 처음 요소를 삭제합니다.

const arrNum1 = [100,200,300,400,500];
const arrPush1 = arrNum1.unshift(600);

console.log(arrNum1) // [600,100,200,300,400,500]

const arrNum2 = [100,200,300,400,500];
const arrPush2 = arrNum2.shift();

console.log(arrNum2) //[200,300,400,500]
